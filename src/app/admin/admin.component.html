Application Settings

Auto-Save

When enabled, data is persisted automatically on every change. When disabled, users must manually click the Save button to persist changes.

Player Status Icons

When enabled, star (⭐) and reserve indicators are included when copying match data to the clipboard.

Clipboard Player Order

When enabled, players within each team are listed in a random order instead of by rating when copying match details to the clipboard.

Default Match Schedule

These day/time pairs are used when cycling through options with the Next Game Day button in the game events form. The time is inserted as the match suffix.
